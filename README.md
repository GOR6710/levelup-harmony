# LevelUp HarmonyOS

LevelUp Skill System - HarmonyOS ArkTS 版本

## 功能特性

- ✅ 用户登录/注册
- ✅ 任务管理
- ✅ 属性面板
- ✅ 等级与经验值系统
- ✅ 深色主题 UI

## 技术栈

- HarmonyOS NEXT (API 12+)
- ArkTS
- DevEco Studio

## 快速开始

### 环境要求
- DevEco Studio 4.0+
- HarmonyOS SDK
- Node.js 18+

### 安装

```bash
# 克隆仓库
git clone https://github.com/GOR6710/levelup-harmony.git

# 用 DevEco Studio 打开项目
# File -> Open -> 选择 levelup-harmony 目录
```

### 运行

1. 连接 HarmonyOS 设备或启动模拟器
2. 点击 DevEco Studio 的 Run 按钮
